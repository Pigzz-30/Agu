# 炒股秘籍每日复盘

本仓库用于归档自动生成的 A 股每日复盘与开盘前消息简报。

## 手机查看

GitHub Pages 展示页，启用后可访问：

https://pigzz-30.github.io/Agu/

首次启用步骤：

1. 打开仓库 `Settings`
2. 进入 `Pages`
3. `Build and deployment` 选择 `Deploy from a branch`
4. `Branch` 选择 `master`，目录选择 `/ (root)`
5. 保存后等待 1-2 分钟

## 输出目录

- `每日复盘/YYYY-MM-DD/`
  - `YYYY-MM-DD_9点_A股消息简报.md`
  - `YYYY-MM-DD_9点_A股消息简报.png`
  - `YYYY-MM-DD_15点_A股收盘复盘.md`
  - `YYYY-MM-DD_15点_A股收盘复盘.png`

## 筛选规则

- 个股只筛选主板股票：`000` / `001` / `002` / `003` / `600` / `601` / `603` / `605`
- 剔除科创板与创业板个股：`688` / `300` / `301`
- ETF 可正常纳入观察池

## 说明

内容仅作研究复盘，不构成投资建议。
